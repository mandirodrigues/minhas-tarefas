Minhas Tarefas 📝

Aplicação simples de lista de tarefas desenvolvida com HTML, CSS e JavaScript, que permite adicionar, remover e visualizar tarefas diretamente no navegador.

Funcionalidades

- Adicionar novas tarefas

- Excluir tarefas concluídas

- Contador de tarefas

- Interface responsiva e intuitiva

- Fonte personalizada (Google Fonts – Dancing Script)

Conceitos praticados

- Estruturação de página com HTML5

- Estilização com CSS3

- Lógica e manipulação de elementos com JavaScript

- Organização de código e boas práticas

- Uso do Git e GitHub para versionamento

💻 Tecnologias utilizadas

- HTML5

- CSS3

- JavaScript
